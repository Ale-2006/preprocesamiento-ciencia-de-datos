# DOCUMENTACION DEL PROYECTO

## Introduccion
En este proyecto tiene como objetivo aplicar tecnicas de control de versiones usando Git y GitHud, ademas de implementar un proceso de preprocesamiento da datos utilizando Python y Pandas.

---

## Comandos Git Utilizados

### git clone
Permite clonar un repositorio remoto.

### git add .
Agrega archivos al area de preparacion.

### git commit
Guarda los cambios realizados.

### git push
Sube cambios al repositorio remoto.

### git checkout -b
Crea una nueva rama.

### git merge
Fusiona ramas del proyecto.

---

## Automatizacion
GitHub Actions esto permite automatizar tareas dentro del repositorio como pruebas y validaciones automaticas.

---

## Evidencias
Se incluyen capturas de:
- Comandos ejecutados.
- Pull Request creada.
- Fusion de ramas.
- Repositorio funcionando correctamente.
